\doxysubsection{Detailed Description}
Simulates a web server that processes web requests. 

Simulates a web server that processes requests.

The \doxylink{class_web_server}{Web\+Server} class is responsible for handling incoming web requests, processing them, and tracking how long it remains idle.

The \doxylink{class_web_server}{Web\+Server} class is responsible for managing the processing of web requests. It tracks whether it is busy, the time remaining for the current request, and how long it has been idle.
